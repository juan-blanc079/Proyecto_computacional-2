Simulación Numérica de Funciones de Onda para el Átomo de Hidrógeno¶

Este proyecto tiene como objetivo construir numricamente las funciones de onda del electrn en el potencial de Coulomb unidimensional (modelo simplificado del tomo de hidrgeno) utilizando la ecuacin de Schrdinger independiente del tiempo. La simulacin se basa en el mtodo de disparo (shooting method) y la integracin numrica para ajustar el valor de la energa hasta obtener una solucin con la propiedad asinttica deseada.

Fundamento Teórico¶

La ecuacin de Schrdinger independiente del tiempo para un electrn en una dimensin se puede expresar como:

$$ -\frac{\hbar^2}{2m}\frac{d^2\psi(x)}{dx^2} + U(x)\psi(x) = E\psi(x) $$

donde:

  • $\psi(x)$ es la funcin de onda.
  • $E$ es la energa (el valor propio que se busca para cada estado estacionario).
  • $U(x)$ es la energa potencial.
    En el caso del tomo de hidrgeno, el potencial de Coulomb para un electrn en presencia de un protn se aproxima como:
$$ U(x) = -\frac{D}{x} $$

con $D$ siendo una constante relacionada a la fuerza de atraccin electrosttica.

Método de Disparo (Shooting Method)¶

El mtodo de disparo consiste en:

  1. Iniciar la integracin de la ecuacin diferencial a partir de una posicin cercana a (x = 0) (se evita el cero exacto para no enfrentarse a la singularidad del potencial).
  2. Elegir condiciones iniciales, por ejemplo:
    • $\psi(0)=0$
    • $\psi'(0)=0.5$
  3. Disparar (integrar) con un valor inicial de la energa $E$ y ajustar este valor de modo que, al integrar, la funcin de onda $\psi(x)$ cumpla la condicin asinttica de decaer a medida que $x \to \infty$.

import numpy as np
import matplotlib.pyplot as plt
from numpy import pi

# Constantes
C = 26.24
D = 1.44
tiempo = np.linspace(0, 1000, 10000)
dx = 0.001

def potential_energy(x):
    # Se evita la singularidad en x=0; aunque se contempla que nunca se evala en exactamente 0
    return -D/x if x != 0 else 0

def wave_function(particle):
    history = []
    for i in range(len(tiempo)):
        x = particle["x"]
        yi_0 = particle["y_0"]
        yi_1 = particle["y_1"]

        # Se guarda el par (x, )
        history.append([x, yi_0])
        
        E = particle["energy"]
        U = potential_energy(x)
        
        # Ecuacin de Schrdinger reescrita en forma diferencial de 2 orden:
        y_2 = -C * (E - U) * yi_0
        y_1 = yi_1 + y_2 * dx
        y_0 = yi_0 + y_1 * dx
        
        particle["y_0"] = y_0
        particle["y_1"] = y_1 
        particle["x"] = x + dx
        
    particle["history"] = history
    return particle

def graphs(initial_conditions, energy_levels):
    # Para evitar problemas con la singularidad en x = 0, iniciamos la integracin en x = dx/2
    x0 = dx/2  
    psi0 = initial_conditions[1] * x0  # Aproximacin: (x0)  '(0)*x0
    psi1 = initial_conditions[1]
    x_lims = [1.25, 1.8, 2.5, 3.5]
    
    for i, energy in enumerate(energy_levels):
        plt.figure(figsize=(10, 8))
        plt.title(f"Funcin de Onda para E = {energy} eV")
        plt.xlabel("x (m)")
        plt.ylabel("(x)")
        plt.ylim(-10, 10)
        plt.xlim(-0.01, 10)
        plt.grid(True)
        
        # Definir la partcula con las condiciones iniciales modificadas
        particle = {
            "id": i,
            "x": x0,
            "energy": energy,
            "y_0": psi0,
            "y_1": psi1
        }
        
        particle = wave_function(particle)
        history = particle["history"]
        xs = [p[0] for p in history]
        ys = [p[1] for p in history]
        plt.plot(xs, ys, lw=1, label=f"E = {energy} eV")
        plt.legend()
        plt.ylim(-0.05, 0.05)
        plt.xlim(-0.05, x_lims[i])
        plt.show()

# Iniciacin
initial_conditions = np.array([0, 0.5])  # [(0), '(0)]
energy_levels = np.array([-13.606461780325, -3.401236, -1.511589, -0.85025])

graphs(initial_conditions, energy_levels)
